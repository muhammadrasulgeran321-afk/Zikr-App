# Zikr Counter — Build APK via GitHub (Free, No installs needed)

## ⚡ Quickest way to get the APK (5 minutes, free)

### Step 1 — Create a free GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** button (top right) → **New repository**
2. Name it: `ZikrApp`
3. Set to **Public**
4. Click **Create repository**

### Step 3 — Upload the project files
1. On your new repo page, click **uploading an existing file**
2. Drag and drop ALL files and folders from this ZIP (keep the folder structure)
3. Scroll down, click **Commit changes**

### Step 4 — Watch it build automatically
1. Click the **Actions** tab on your repository
2. You'll see **"Build APK"** running — wait ~3 minutes
3. When it shows a green ✅, click on it
4. Scroll down to **Artifacts** section
5. Click **ZikrApp-debug** to download your APK!

### Step 5 — Install on your Android phone
1. Send the APK to your phone (WhatsApp, email, Google Drive, etc.)
2. On your phone: **Settings → Security → Unknown Sources → Allow**
   (or "Install unknown apps" depending on your Android version)
3. Open the APK file and tap Install
4. Done! Look for the Zikr Counter app with the crescent icon 🌙

## App Features
- Create unlimited custom Zikr sequences
- Set any count per dua (5, 33, 100, 165, etc.)
- Add Arabic text for each dua
- Tap to count — auto-advances to next dua
- Vibration on tap (can be turned off)
- Vibration on dua complete (can be turned off)
- Sound toggle (beep on tap, melody on dua done)
- Round counter (tracks per session)
- All-time stats: total rounds, total count, sessions
- Reset button
- Skip button
